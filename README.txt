Genera firmas personalizadas para autores en WordPress, permitiendo a cada autor tener una firma única al final de sus posts.

== Descripción ==

El plugin "Autor Firmas Personalizadas" permite a los administradores de WordPress configurar firmas personalizadas para cada autor del sitio. Estas firmas se mostrarán automáticamente al final de los posts individuales, añadiendo un toque personal a cada artículo.

Características principales:
* Configuración de firmas personalizadas para cada autor
* Visualización automática de la firma al final de los posts individuales
* Soporte para HTML en las firmas, permitiendo enlaces y formateo básico
* Fácil de configurar y usar

== Instalación ==

1. Sube los archivos del plugin al directorio `/wp-content/plugins/autor-firmas-personalizadas`, o instala el plugin directamente desde el panel de WordPress.
2. Activa el plugin a través del menú 'Plugins' en WordPress.
3. Ve a 'Ajustes' > 'Firmas de Autor' para configurar las firmas de los autores.

== Preguntas Frecuentes ==

= ¿Puedo usar HTML en las firmas? =

Sí, el plugin permite el uso de HTML básico en las firmas, incluyendo enlaces y formateo de texto.

= ¿La firma se muestra en todos los posts? =

La firma se muestra solo en los posts individuales, no en las páginas de archivo o resúmenes.
